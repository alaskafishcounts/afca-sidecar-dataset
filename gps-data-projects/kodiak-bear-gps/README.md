# Kodiak Bear GPS Tracking Data

This directory contains GPS tracking data for Kodiak bears, providing detailed movement patterns and habitat usage information.

## Data Overview

- **File**: `kodiak-bear-gps-data.json`
- **Format**: JSON with GPS coordinates and timestamps
- **Coverage**: Kodiak Island region
- **Species**: Kodiak bear (Ursus arctos middendorffi)

## Data Structure

The JSON file contains GPS tracking points with:
- Latitude and longitude coordinates
- Timestamps for movement tracking
- Bear identification information
- Movement patterns and habitat usage

## Usage

This data can be used for:
- Wildlife movement analysis
- Habitat mapping and conservation
- Integration with AFCA ecosystem data
- Research on bear behavior and migration patterns

### Study Design & Methodology
**Study Period**: 2008-2015 (8-year longitudinal study)
**Sample Size**: 51 female Kodiak bears (Ursus arctos middendorffi)
**Collection Method**: GPS radio collar technology
**Collar Programming**: Hourly location fixes from 15 May through 15 November
**Study Focus Period**: 10 July through end of August (peak sockeye salmon run)
**Total GPS Records**: 172,545 individual tracking points

### Technical Collection Details
**Collar Technology**: 
- Ultrahigh frequency (UHF) radio transmitter
- GPS positioning system with hourly fix capability
- Activity level monitoring sensors
- Temperature monitoring capability
- Position Dilution of Precision (PDOP) tracking

**Data Retrieval Method**:
- Airplane-mounted UHF receiver system
- Aerial data download from collars
- Ground-based verification and calibration
- Quality control protocols for GPS accuracy

**Data Quality Metrics**:
- PDOP values recorded for accuracy assessment
- Temperature data for environmental context
- Activity level measurements for behavioral analysis
- Timestamp precision to the second
